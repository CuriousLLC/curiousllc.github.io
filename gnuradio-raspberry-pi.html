GNURadio Raspberry Pi

Posted on Thu 20 August 2015 in gnuradio raspberrypi

On the Raspberry PI 2, install some required components

sudo apt-get install libuhd-dev
sudo apt-get install libboost-dev
# prolly qt and gnuradio as well

See github for example project that now works, written in PySide. Also example code written in C++. Now to improve the workflow with a xcompiler. Maybe a new article?

Setup cross compiler and libraries

mount sshfs
sudo ln -s /home/rday/Development/raspberrypi/mntrpi/lib/arm-linux-gnueabihf/ /lib/arm-linux-gnueabihf
sudo ln -s /home/rday/Development/raspberrypi/mntrpi/usr/lib/arm-linux-gnueabihf/ /usr/lib/arm-linux-gnueabihf
cp -R /usr/share/qt4/mkspecs/linux-arm-gnueabi-g++ ./mkspec
# Update with changes
qmake -spec mkspec/
make
